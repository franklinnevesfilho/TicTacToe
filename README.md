# Tic Tac Toe
This is a simple implementation of the classic Tic Tac Toe game in Python. 
The game is played by a single player against the AI.

## implementation
The AI uses the Minimax algorithm along with alpha-beta pruning to make optimal moves.

## How to play
1. Run the main.py file.
2. Select the player symbol (X or O).
3. play the game


## Requirements
- Python 3.x
- No additional libraries are required.

## How to run
1. Clone the repository.
2. Navigate to the project directory.
3. Run the command `python main.py` to start the game.
